# ガチャ カスタムアソート シミュレータ v2

自分のPCにある画像（サムネイル）を割り当てて、ガチャガチャの**アソート（中身の内訳）**を再現し、
「実際に回したらどの順番で何が出るか」を疑似体験できるシミュレータです。

HTML / CSS / JavaScript のみの**単一ファイル**で動作します。サーバー・インストール・ネット接続は不要です。

## 使い方

1. [`gacha_sim_custom_upload_v2.html`](gacha_sim_custom_upload_v2.html) をダウンロードする
2. ブラウザ（Chrome / Edge / Firefox / Safari）でファイルを開く
3. 左パネルでアソートを設定して「シミュレート開始 ▶」

詳しい仕様は [`gacha_sim_custom_upload_v2_解説.html`](gacha_sim_custom_upload_v2_解説.html) を参照してください。

## 主な機能

- 種類数 1〜24種、タイプごとに名前・初期数・排出済み数を自由に設定
- 各タイプに自分のPCの画像を割り当て可能（自動縮小してブラウザ内に保存）
- 現実のガチャと同じ**非復元抽選**（引いたら減る・最後まで引けば内訳どおり）
- 10連ガチャ／1回戻す（アンドゥ）／履歴のCSV保存
- 残り数バーと「次の1回で出る確率」のリアルタイム表示
- 設定は自動保存され、次回開いたとき復元される

## プライバシー

アップロードした画像・設定は**自分のブラウザの localStorage に保存されるだけ**で、外部には一切送信されません。

## 利用上の注意

商品サムネイルなどの画像には著作権があります。
本アプリは**自分のPCにダウンロードして、個人で楽しむ範囲**でご利用ください。
このリポジトリに画像ファイルは含まれていません（画像なしでもカラーパネル表示で動作します）。
